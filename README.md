## 4. Installation ❤️‍🔥

#### Clone the repo

```sh
$ git clone https://github.com/preetamvarun/Restaurant-App.git
```
### Navigate to project directory

```sh
$cd Restaurant-App/
```

### Install dependencies 
```sh
npm install
```

Start the app
```sh
npm run start
```

#### To create a build 
```sh
npm run build
```
